# /deep — Thorough Investigation

Deep dive into complex issues.

## Usage
```
/deep investigate why tests fail intermittently
/deep security audit the auth module
/deep performance profile the API endpoints
/deep 이 버그의 근본 원인 분석
```

## Behavior

1. Define investigation scope
2. Systematic evidence gathering
3. Form and test hypotheses
4. Identify root cause with evidence
5. Recommend targeted fix + prevention

- Agent: reasoner (Codex via codex-shell MCP)
